## [763. Partition Labels](https://leetcode.com/problems/partition-labels/)

### First Solution
**Java**

```java
class Solution {
    public List<Integer> partitionLabels(String s) {
        Map<Character, Integer> char2lastPos = new HashMap<>();
        int n = s.length();
        for(int i = 0; i < n; i += 1) {
            char c = s.charAt(i);
            int lastPos = Math.max(char2lastPos.getOrDefault(c, 0), i);
            char2lastPos.put(c, lastPos);
        }

        int left = 0, right = 0;
        List<Integer> ans = new ArrayList<>();
        for(int i = 0; i < n; i += 1) {
            char c = s.charAt(i);
            right = Math.max(right, char2lastPos.get(c));
            if(i == right) {
                ans.add(right - left + 1);
                left = right = right + 1;
            }
        }
        return ans;
    }
}
```

**Python**
```python
class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        last_to_last_pos = defaultdict(int)
        for i, c in enumerate(s):
            last_to_last_pos[c] = max(last_to_last_pos[c], i)
        
        left, right, ans = 0, 0, []
        for i, c in enumerate(s):
            right = max(right, last_to_last_pos[c])
            if i == right:
                ans.append(right - left + 1)
                left = right = right + 1
        return ans
```
