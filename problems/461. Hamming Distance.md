## [461. Hamming Distance]()

### First Solution
**Java**
```java
class Solution {
    public int hammingDistance(int x, int y) {
        int res = 0;
        while(x != 0 || y != 0) {
            if(x % 2 != y % 2) {
                res += 1;
            }
            x >>= 1;
            y >>= 1;
        }
        return res;
    }
}
```
